# Dandeli Travel AI Agent
An intelligent generative AI proxy designed to automate resort research, ranking, semantic intent extraction, and WhatsApp Twilio booking workflows for properties in Dandeli, Karnataka.

### Tech Stack:
- **Frontend**: Next.js 15, TailwindCSS, Clerk
- **Backend**: FastAPI, Uvicorn
- **AI Core**: LangGraph Multi-Agent Orchestrator (Booker, Planner, Researcher)
- **Vector Engine**: ChromaDB
- **LLMs**: Google Gemini GenAI & Llama Instant Groq
